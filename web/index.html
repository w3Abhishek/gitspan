Setup & Deployment

1. Production Docker Execution

GitSpan is designed to be hosted on any Linux VPS (Ubuntu/Debian) using Docker. The Compose structure provisions your database alongside internal networks seamlessly.

git clone https://github.com/w3Abhishek/gitspan.git cd gitspan docker-compose up -d --build

2. OAuth Key Configuration

Navigate to GitHub/GitLab's developer settings, obtain your Client IDs, and inject them into frontend/.env and backend/.env. Set your GitHub Auth Callback URL to: https://[your-domain]/api/auth/callback/github
